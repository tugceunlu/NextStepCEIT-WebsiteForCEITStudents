CEIT211 Programming Languages II

This course provides an introduction to the fundamental concepts and principles of Java programming, with an emphasis on object-oriented programming (OOP). It covers essential topics such as data types, variables, decision structures, loops, and methods. Additionally, advanced topics like inheritance, polymorphism, abstract classes, graphical user interfaces (GUIs), and database connectivity will be explored. Through hands-on projects, students will learn to develop robust Java applications for desktop environments, enhancing their problem-solving and coding skills.

    Course Objectives

  • Introduce fundamentals, including decision structures, loops, and methods in Java
  • Understanding of object-oriented programming concepts and their applications in Java
  • Equip design and develop Java applications with user interfaces and database connectivity
  • Enhance problem-solving skills by applying Java programming principles to real-world scenarios
